# JIDARAKU

## ◆開発者
TK Lab.  

Twitter : @TK_Lab_  

## ◆開発環境  
Windows10 Pro 64bit  

Visual Studio Community 2022  

## ◆ビルド方法
１．スタートメニューからVisual Studio の「開発者用コマンドプロンプト」を起動

２．カレントディレクトリをクローンしてきたディレクトリに移動

３．開発者コマンドプロンプトから make.cmd を実行

## ◆起動方法  
出来上がったJIDARAKU.exe をコマンドプロンプトから実行  

または、エクスプローラー上でダブルクリック

## ◆終了方法  
ウィンドウの×ボタンを押す  

またはJIDARAKU.exeのウィンドウを選択して Ctrl + C

## ◆ 注意事項
もともとが英語配列環境であることをを前提とした実装となってるので、日本語配列環境ではそのままだとうまく動かないと思います。
